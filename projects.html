My Projects

One of my recent challenging projects involved creating a music recommendation application using Python and a Spotify dataset. My team and I developed an algorithm to generate song recommendations based on user input, including preferences like danceability, popularity, genre, and artist. The application created personalized playlists by aligning user preferences with the data.

One challenge we tackled was converting technical variables into more user-friendly terms. For example, we transformed the danceability metric from a 0-1.0 scale to a 1-100 scale to improve usability. This project taught me valuable lessons about both the technical and user-experience aspects of building an application.

For a project in my databases course, my team and I worked on designing and implementing a database for a company based on a detailed case study.

  • Conceptual Design: Identified key entities and relationships and created an entity relationship diagram as a blueprint.
  • Logical Design: Normalized the data to reduce redundancy and ensure accuracy by organizing attributes and relationships into structured tables.
  • Physical Implementation: Built and tested the database in Microsoft SQL Server, ensuring functionality and performance.

This project helped me understand the importance of a well-structured database and careful planning to solve complex problems effectively.
